# CaveExplorer - SFML Game

<h3>
    <p align="center"> <i>1) Configure and generate binaries</i> </p>
</h3>

    cmake -S . -B build

<h3>
    <p align="center"> <i>2) Build object files and link executable</i> </p>
</h3>
    
    cmake --build build
    
<h3>
    <p align="center"> <i>3) Run executable</i> </p>
</h3>

    ./CaveExplorer
